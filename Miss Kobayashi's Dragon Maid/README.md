# Miss Kobayashi's Dragon Maid
Based on **The0x539**'s release, which you will need in order to use these scrubs.  

The majority of the changes are matching the episode titles to the official release.  
I know the official release is controversial, but I think the episode titles are overall a bit more accurate.  
Since it's a matter of translation, it's somewhat subjective, of course.  

Empty lines were removed.  
It's possible the comments can be removed too but I'll have to watch through to check for sure.  

In episode 9, "sports meet" was changed to "sports festival" to match the official translation.  
Likewise, episode 13 had all instances of "chaos" changed to "demise".  
I'm not convinced these are "better" translations, but since I matched the titles to the official release, these changes were required.  

Episode 1 of the Dragon Something specials has an interesting change.  
The original subs had "Do-Re-Me-Fa-So-La-Ti-Do". This is wrong - "Me" should be "Mi", and the Japanese subtitles confirm this.  
However, [solfĂ¨ge](https://en.wikipedia.org/wiki/Solf%C3%A8ge) is different between English and Japanese.  
"Sol" typically becomes "So", else it would be two syllables in Japanese, so I decided to keep the Japanese version.  
"Ti" is written as "Shi", which is somewhat confusing, as the syllable "Si" can be used in place of "Ti".  
Based on what I could gather from [Kentaro Sato's 'Sato Method'](http://www.wisemanproject.com/edu&res-solfĂ¨ge-j.html), it seems "Ti" is the most common romanisation for the Japanese solfĂ¨ge.  
Therefore, despite my original change to "Si", I decided to change it back to "Ti".  

There are a few miscellaneous lines which were slightly changed by another scrubber but I can no longer find their work to credit them.  
Said other scrubber supposedly fixed some issues, but introduced many others, so I merged their changes into my scrubs to keep only the good changes.  

All the episodes have had the Japanese titles added to the metadata, so you'll see the Japanese names by default when playing them.  
I like to do this for archival purposes - it leaves the original name intact alongside the subbed files, so you can reference them if necessary.
