# Kamen Rider X
This is my scrub of existing subtitles for Kamen Rider X.
They are based on the Turn Up Scrubs and The Masked Subbers releases.

The following changes have been made:
* Changed all of instances of "Masked" to "Kamen".
* Subs for the first 6 episodes from Turn Up Scrubs now match the formatting from the TMS releases (episodes 7-35).
* The logo at the start of each episode has been subbed in a matching style.  
They're colour-matched to each episode too, since the colour grading changes between episodes (probably due to being film transfers).
* Retimed all logos, title cards, signs, and other on-screen text to be frame-perfect (hopefully).
* Signs were repositioned and reformatted, with a select few (mainly on still shots) matching the formatting of the originals.
* OP and ED subtitles retimed with the romaji copied from the first 6 episodes.  
Minor translation changes were made to keep the meaning but improve the English.
* Names have been made consistent with the Turn Up Scrubs (e.g. Ryoko -> Ryouko).  
This means the inclusion of long vowels where they exist.  
The Masked Subbers had inconsistent use of long vowels, so I figured this was the easiest solution (and it's more accurate).
* All instances of "G.O.D." (primarily in episodes 1-6) have been changed to "GOD".  
Though the former is usually used in English, and it's more accurate for an acronym, it's always written as the latter in Japanese media when written with English characters.
* Trivia/translation notes were removed from all bumpers.  
Some of them were interesting, but they're not part of the show.  
My personal opinion is that these belong in accompanying material, not as part of the in-show subtitles.
* In episode 23, TMS mentioned a "mysterious" line from a little girl. Their translation is wrong.  
Checking the Japanese subtitles, the line in Japanese is 「私 うち中で軽井沢へ行くの」.  
Therefore, the line "My family's going to Mt. Karei!" should be something like "I'm going to my house in Karuizawa".  
Karuizawa is resort town in Nagano, so it's clearly her family's holiday home.  
This makes more sense than the by-ear guess translation TMS did.
* Episode titles were added to the filenames.
* Fonts have been made consistent between episodes.

Timing issues likely still exist - I haven't checked every single line.

You can rename these subs to use them with the the original files, or use the included patches.  
If I can find the original raws (probably from BunnyHat), I will provide patches for those too.
