# Name Changes

Here are the name changes and my reasoning behind them.  
This isn't an exhaustive list, at least for now - I just wrote down what I consider to be some of the more controversial/confusing names whilst scrubbing.  

### Monsters

* カニバブラー Crabubbler -> Crab Bubbler  
Despite being written in katakana, the Japanese name can be literally translated, since "kani" is "crab" and "bubbler" is already in English.  
However, the original English name effectively turns into a pun. Maybe this was intended in Japanese, maybe not, I'm not sure.  
After a few days going back and forth on this, I think "Crab Bubbler" is the best middle-ground.  
* ドクガンダー Poison-Mothdra -> Dokugander  
The English name just doesn't make sense. Whilst "doku" means "poison", Dokugander has both a larva form and a moth form.  
That means the English name serves as both a spoiler and an incorrect name at one point.  
* アマゾニア Amazonha -> Amazonia  
This one was just wrong, end of story.  
* 地獄サンダー Antlithunder -> Jigoku Thunder  
There's no clean way to translate this one. An antlion is "arijigoku" ("アリジゴク" or "蟻地獄"), but only part of that name is used here.  
"Thunder" is straight-forward. Also, "jigoku" can mean "hell". It's probably an intentional pun... which doesn't work in English.  
Maybe it's named as such because of the hellish appearance of the real deal.  
Antlions are freakish and belong in hell - is that what Ishinomori was trying to tell us?  
Anyway, I've left the Japanese name intact.  
* ムカデラス Centiperas -> Mukadelas  
The English name is made up, trying to copy the Japanese name's mutation of "mukade" (centipede) with an additional suffix.  
If I were to make up a name, "Centipederas" is probably closer, but it's still nonsense.  
As for the suffix, it's written in katakana, so I have no idea if it means anything.  
Whether it's Romanised as "ras" or "las" seems subjective unless I learn otherwise.  
* アリキメデス Antchimedes -> Arichimedes  
This is one that I'm split on. On the one hand, "ari" can be literally translated to "ant".  
On the other hand, the rest of the name appears to be a pun on "Archimedes", but it's only part of the name.  
The pun still works in English, but since it both works slightly better as a pun if left in Japanese, and it's a made-up name, I kept the original name.  
* 毒トカゲ男 Bearded Dragon (Man) -> Beaded Lizard Man  
A Bearded Dragon is not poisonous. The Japanese name, "Dokutokage" (literally "poisonous lizard"), is a family of poisonous lizards known as Beaded Lizards in English.  
How "Bearded Dragon" ever got used is beyond me - it's completely incorrect.  
Also, there was inconsistent use of "Man" in the original name.  

Gel Shocker names get a bit trickier. Despite using katakana, they're often directly translatable due to just smashing two animal names together.  
Sometimes they are only a letter or two off from being directly translatable, so I've handled them on a case-by-case basis for now.  
* ガニコウモル Crabbat -> Ganikomoru  
An instance of two "corrupted" Japanese words mixed together. It's not quite "Crab Bat", so I've just left this one as the Japanese name.  
* サソリトカゲス Scorpiolizard -> Scorpiontokagesu  
"Sasori" can literally be translated to "scorpion", but "tokagesu" adds another syllable to the word for "lizard", "tokage".  
I'm not sure how best to handle these names that can be literally translated for one half but not the other, so I've just translated the half that can be translated.  
* ネズコンドル Ratcondor -> Nezcondor  
This is another tricky name. Rat/mouse is "nezumi" (ネズミ), so the English name doesn't really convey the truncated Japanese name.  
I could have made it something like "Racondor", but then I'm ultimately making up names, which is what this scrub is trying to ultimately avoid.  
* ムカデタイガー Centigerpede -> Centipede Tiger  
The Japanese name is directly translatable. The English name is unnecessarily complicated.  
There's a bit of an inconsistency here since I kept "Mukadelas" previously.  
See that entry for my justification on that choice.  
* ナメクジキノコ Slugshroom -> Slug Mushroom  
Another directly translatable name that was unnecessarily shortened in English.  
The English name definitely rolls off the tongue better... and is ultimately less accurate.  
* ガラオックス Crowox -> Garaox  
Another name where only part of the first word is used in the Japanese name, "karasu" for "crow" (or "garasu" when added to the end of other names of crow species).  
Maybe "Crox" would be a more accurate English name, albeit still made-up... and now sounding like a crocodile monster (or the worst footwear you can buy).  
* ハエトリバチ Flytrabee -> Flytrap Bee  
This is fairly straight-forward but needs to be broken down.  
"Haetoribana" is a venus flytrap, so "haetori" is for "flytrap" by itself.  
This might be cheating - I haven't seen "haetori" written completely alone.  
"Flytrap" seems to be written as "haetorisou" ("ハエトリソウ") - note the extra suffix, but it should technically work.  
"Bachi" is "bee", but often written as "hanabachi".  
Since "hana" is "flower", and there are other types of bees with a different prefix, I think it makes sense to break it down to "Flytrap Bee".  
* エイドクガー Raypoimoth -> Raydokuga  
I hate this name - it's a mess in both languages. "Ei" is "ray", that's simple.  
"Dokugaa" is more complicated. "Ga" is "moth", "doku" is poison.  
There's a moth called the "scarce vapourer" - "akamondokuga" (アカモンドクガ) in Japanese - but it's not an exact match.  
I don't like doing a partial translation, but since "ei" and "ray" are almost identical anyway, I figure this is about the best I can do without making up an English name.  
Since I removed long vowels everywhere else, they're removed here too.  
I might still change this one, because I'm unhappy with it, but I'm not sure.  
* ヒルカメレオン Leechameleon -> Leech Chameleon  
Again, the English name is contracted into a pun. This might work "better" in English... but it's wrong.  
This one feels especially agregious - do you pronounce the "ch" as a hard sound or a soft sound? Making them separate words solve this conundrum.  
"Hiru" translates directly to "leech", and "chameleon" is the same in both languages (save for the obvious differences).
