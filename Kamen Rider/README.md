# Kamen Rider (1971)

Based on the SoftSHOUT release, which uses Shout! Factory's subtitles.  

I wasn't happy with any of the releases available so I decided to make my own.  
Shout! Factory has an incessant habit of using English name order, which doesn't line up with the audio. Their subtitles are otherwise mostly decent.  
The 1080p release with XIG's subs has titles cut off and unnecessarily large text, along with honorifics which I prefer to translate/remove for a translation.  
Since it's almost impossible to tell the difference between the 1080p and 720p encodes, I opted for editing the Shout! subs because the 720p files save space whilst looking pretty much identical.  

This is very much still a work-in-progress, as I have yet to watch it through with the changes.  
I'll make changes if/when necessary once I get around to watching these scrubs.  
Patches will likely come after I do so. Until then, I haven't bothered making them.  

Changes:
* Changed name order back to Japanese (last name, first name).  
* Changed monster names back to their Japanese versions when a direct translation (e.g. Kumo Otoko -> Bat Man) can't be made.  
This is typically when the monster's name is an amalgamation of words or just completely made-up.  
I'm not a fan of translating proper nouns in general, but leaving some of the names in Japanese when they are rather literal in both languages seems pointless.  
Equally pointless, in my opinion, is making up new names to translate already-made-up names.  
Sometimes it works fine (e.g. Kanibubbler -> Crabubbler), most some of them are ridiculous (e.g. Dokugander -> Poison-Mothdra).  
I might make notes on every individual monster name choice in the future, but it's rather time-consuming.  
* Tachibana's name was changed from "Tobee" to "Tobei". This is more or less a subjective change.  
* "Doctor Death" was changed to "Doctor Shinigami", since this isn't a direct translation.  
* Removed long vowels in names. I might change this back in the future depending on how I decide to match other Rider shows I scrub.
* Changed "motorcycle" to "motorbike" because I think the single syllable works better for the Japanese word "Autobi" (Ńé¬Ńā╝ŃāłŃāÉŃéż).  
* Changed episode titles where appropriate to match other changes.
